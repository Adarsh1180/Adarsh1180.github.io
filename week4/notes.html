HTML Tags

<p>
Defines a paragraph of text.
<a>
Creates a hyperlink, linking to another webpage or resource.
<img>
Embeds an image into the HTML document.
<ul> and <li>
Creates an unordered list, where each item is represented by the <li> tag.
<div>
Defines a division or section in an HTML document, often used for layout purposes.
